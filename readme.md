# Learning Node 

## Lesson 1 Intro to NPM

### help

```bash
npm -h                  ; show in cli
npm [command] -h        ; show about command in cli
npm help [command]      ; opens browser
npm help-search [term]  ; search help content for the term

```

```bash
npm init                ; initialize packagae.json with initial values

```
